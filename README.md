## Overview

This project documents my learning progression through CSS fundamentals, covering everything from basic styling to advanced layout techniques like Flexbox and CSS Grid. Each lesson includes:

- **Index files**: Interactive examples demonstrating concepts
- **Exercise files**: Practice problems to apply learning
- **Solution files**: Reference implementations

## Lessons Covered

### Section 5: CSS Basics
- **5.1 Adding CSS** - Ways to include CSS in HTML (inline, internal, external)
- **5.3 CSS Selectors** - Element, class, ID selectors and combinations
- **5.4 Color Vocab Project** - Real-world color styling application

### Section 6: Styling Properties
- **6.0 CSS Colors** - Color models and implementations
- **6.1 Font Properties** - Typography and font styling
- **6.3 CSS Box Model** - Margins, padding, borders, content areas
- **6.4 Motivation Meme Project** - Practical project using text and color styling

### Section 7: Advanced Selectors & Positioning
- **7.0 CSS Cascade** - Understanding CSS specificity and inheritance
- **7.1 Combining Selectors** - Advanced selector techniques
- **7.2 CSS Positioning** - Static, relative, absolute, and fixed positioning
- **7.3 Flag Project** - Creating flags using positioning techniques

### Section 8: Display & Layout Basics
- **8.0 CSS Display** - Block, inline, and inline-block elements
- **8.1 CSS Float** - Float-based layouts
- **8.2 Responsiveness** - Bootstrap, Flexbox, and Grid introduction
- **8.3 Media Query** - Responsive design techniques
- **8.4 Web Design Agency Project** - Full-page responsive website

### Section 9: Flexbox
- **9.0 Display Flex** - Flexbox fundamentals
- **9.1 Flex Direction** - Controlling flex container direction
- **9.4 Flexbox Pricing Table Project** - Building a pricing table with Flexbox

### Section 10: CSS Grid
- **10.0 Display Grid** - CSS Grid basics
- **10.1 Grid Sizing** - Fixed, fractional, and auto sizing with minmax()
- **10.2 Grid Placement** - Positioning items on the grid
- **10.3 Mondrian Project** - Recreating art using CSS Grid
